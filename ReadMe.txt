ISTRUZIONI PER LA CORRETTA ESECUZIONE DEL PROGRAMMA
    1. Spostarsi attraverso il terminale nella cartella principale del progetto (.../adasGallicoliPernice)
    2. Scrivere la linea di comando make
    3. Spostarsi, attraverso il terminale nella cartella bin/interface
    4. Eseguire il programma hmi specificando la modalità
        4a. ./hmi NORMALE
        4b. ./hmi ARTIFICIALE

NB: Il programma è stato testato, con successo, sui terminali gnome e konsole; si consiglia, tuttavia, di utilizzare gnome, in quanto quest'ultimo è il terminale che viene aperto per la gestione dell'input della Human Machine Interface
